# DSL Programacion Profesional

### Modo de uso

1) Compilar:	

```sh
$ ghc -o main Main.hs
```

2) Crear archivo:

```sh
Texto (Punto(2.5, 1.0), "Hola Hatex");
Rectangulo(Punto(0.0, 0.0), 2.0, 5.0);
Circulo(Punto(0.0, 0.0), 6.0)
```

3) Ejecutar:

```sh
$ ./main archivo
```
